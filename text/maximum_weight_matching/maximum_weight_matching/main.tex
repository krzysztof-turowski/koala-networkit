% Abstract
% The graph matching problem is among the most extensively researched topics in combinatorial optimization. Matching algorithms have been used as subroutines of various optimization algorithms. In this work, we explore some notable algorithms for maximum weight matching on general graphs, discussing how work and how to implement them. We provide implementations, along with performance comparisons based on test results.
